# LexiAssist

LexiAssist is a JavaFX desktop writing assistant that provides autocomplete and spelling suggestions while typing.

## Planned Features

- Load text from a `.txt` file
- Clear editor content
- Autocomplete suggestions
- Spelling suggestions
- Caret-based suggestion popup
- Sidebar suggestion panel

## Technology Stack

- Java 17
- Maven
- JavaFX
- JUnit 5

## Project Note

This project is being built independently as a portfolio project. It is inspired by text-processing and dictionary concepts practiced during Java learning, but the implementation, UI, package structure, class design, and tests are created separately.
